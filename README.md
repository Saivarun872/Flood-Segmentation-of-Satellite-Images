# Flood Segmentation of Satellite Images using U-Net

## Overview

This project implements a U-Net-based deep learning model for segmenting flooded regions from satellite images. The objective is to accurately identify flood-affected areas, which can support disaster response and environmental monitoring.

## Objectives

* Segment flooded regions from satellite imagery.
* Train and evaluate a U-Net segmentation model.
* Visualize predicted segmentation masks.
* Understand the application of deep learning in remote sensing.

## Technologies Used

* PyTorch
* OpenCV
* NumPy
